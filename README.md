## Getting started
1. Clone the repo
```shell
git clone https://github.com/pathull/SunCheck.git
cd suncheck
```

2. Run command in root folder and server folder to install dependencies
```powershell
npm install
```

3. Run front end
```powershell
npm start
```

4. Run server
```powershell
nodemon
```
